## 📦 新增功能

### Message Utils

- 消息解析（命令提取、参数解析）
- @提及提取
- 消息格式化
- 消息截断
- 输入清理

### Command Handler

支持 7 个内置命令：
- `/help` - 显示帮助
- `/start` - 开始新会话
- `/template` - 切换模板
- `/status` - 查看状态
- `/reset` - 重置会话
- `/memory` - 记忆开关
- `/task` - 任务管理

### Workflow Engine

- 工作流定义注册
- 状态机执行（PENDING/RUNNING/WAITING/COMPLETED/FAILED/CANCELLED）
- 顺序步骤执行
- 等待/恢复机制
- 错误处理和重试

### Scheduler Manager

- 三种调度类型：ONCE / INTERVAL / CRON
- 任务状态管理
- 重试机制
- 速率限制
- 任务取消

### Message Polling

- 定时轮询消息
- 消息去重
- 错误重试
- 统计追踪

### Notification Service

- 多渠道支持（Console/Webhook/Email/SMS/Push）
- 优先级过滤
- 速率限制
- 广播功能

## 🔍 技术亮点

### 1. 消息处理管道

实现了完整的消息处理管道：
```
Message → Parser → Command? → Context Loader → 
Agent Router → Task Router → Executor → Formatter → Response
```

### 2. 工作流引擎

支持复杂的工作流编排：
- 顺序执行
- 等待用户确认
- 错误处理
- 状态持久化

### 3. 定时任务调度

灵活的定时任务系统：
- Cron 表达式支持
- 单次/周期任务
- 失败重试
- 并发控制

### 4. 通知服务

多渠道通知系统：
- 动态渠道注册
- 优先级过滤
- 速率限制
- 容错处理
