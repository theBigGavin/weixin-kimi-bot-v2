# 2026年一季度地方政府专项债项目情况分析报告

**报告日期**：2026年3月31日  
**分析时段**：2026年1月1日 - 3月31日

---

## 一、总体发行情况

### 1.1 发行规模创新高

2026年一季度地方政府专项债发行呈现**"靠前发力、提速放量"**的显著特征：

| 指标 | 数据 | 同比变化 |
|------|------|----------|
| 一季度新增专项债发行规模 | **约1.07万亿元** | +41% |
| 1月新增专项债 | 3,677亿元 | - |
| 1-2月新增专项债 | 8,242亿元 | +38.1% |
| 一季度地方债总发行规模 | 突破3万亿元 | 创历史新高 |

**关键数据解读**：
- 截至3月24日，新增专项债发行规模已达**10,712亿元**，突破万亿大关
- 已超过2025年一季度新增专项债总体发行量（约9,602亿元）
- 完成全年4.4万亿元新增限额进度达**18.7%**，快于去年同期

### 1.2 发行节奏前移

财政部提前下达2026年新增债务限额，发行高峰集中在：
- **1月下旬**：年初启动发行
- **3月上旬**：第二波发行高峰
- 全年发行高峰预计出现在**二季度**


### 3.2 重点投向领域详解

#### （1）市政和产业园区基础设施（占比最高）
- **占比**：25%-30%
- **重点方向**：
  - 标准化厂房建设
  - 园区道路及配套管网
  - 产业园区基础设施升级
- **典型案例**：宁波市江北区发行16.6亿元投向产业园区基础设施

#### （2）城市更新领域
- **占比**：25%-30%
- **核心投向**：
  - 城中村改造
  - 老旧小区改造
  - 地下管网建设
  - 棚户区改造

#### （3）交通基础设施
- **占比**：15-20%
- **主要项目**：
  - 高速铁路（如京雄商高铁）
  - 城市轨道交通
  - 高速公路
  - 停车场项目

#### （4）特殊用途专项债
- **土地储备专项债**：重启发行，用于回收闲置存量土地
- **收购存量商品房**：用作保障性住房
- **化债资金**：2万亿元再融资债券用于置换存量隐性债务

## 四、政策机制创新

### 4.1 "负面清单"管理模式

2026年起专项债投向从"正面清单"转为**"负面清单"**：

| 调整项 | 原模式 | 新模式 |
|--------|--------|--------|
| 管理方式 | 正面清单（限定投向） | 负面清单（除禁止外均可） |
| 投向范围 | 11个领域 | 大幅扩容至13大领域、141个细分选项 |
| 地方自主权 | 受限 | 显著提升 |

**禁止投向**：
- 无收益纯公益项目
- 楼堂馆所
- 形象工程与政绩工程
- 非A级景区旅游开发
- 纯商业化产业项目

### 4.2 "自审自发"试点扩围

| 试点范围 | 11个省市 + 雄安新区 |
|----------|---------------------|
| 审核主体 | 省级政府审核批准后即可发行 |
| 申报周期 | 从3-6个月压缩至1-2个月 |
| 试点地区 | 北京、上海、江苏、浙江、广东、深圳等 |

### 4.3 偿债约束强化

- **偿债备付率**：≥1.1（硬性底线）
- **高风险地区**：要求≥1.3
- **禁止事项**：不得以"政府补贴预期""土地溢价分成"等不可验证收益测算

### 4.4 资本金使用优化

- **比例上限**：省级专项债用作资本金比例上限提至**30%**
- **投向扩围**：新兴产业基础设施纳入资本金使用范围
- **"债贷联动"**：推动专项债与银行贷款联动

## 五、市场影响分析

### 5.1 对投资增长的拉动效应

- **早发行早见效**：一季度发行超万亿，快速填补重大项目资金缺口
- **实物工作量**：推动项目加速落地，形成更多实物工作量
- **基建投资**：基建类专项债发行提速，支撑一季度"开门红"

### 5.2 对地方债务风险的化解作用

- **化债资金**：2万亿元再融资债券置换存量隐性债务
- **展期降息**：实现债务展期降息，缓释地方偿债压力
- **腾出空间**：为基建类专项债腾出发行时间和财力空间

### 5.3 发行利率与期限

| 指标 | 数据 |
|------|------|
| 平均发行利率 | 2.19%-2.28%（专项债） |
| 平均发行期限 | 20年左右（专项债） |
| 10年期以上债券占比 | 86%以上 |
| 市场认购倍数 | 平均18倍以上 |

## 六、趋势展望与建议

### 6.1 二季度发行趋势预判

- **发行计划**：已披露二季度计划发行7,813亿元，其中新增专项债2,128亿元
- **发行高峰**：预计全年高峰出现在二季度
- **投向优化**：特殊新增专项债（化债用途）仍有较大发行空间

### 6.2 重点关注事项

1. **"自审自发"试点扩围**：关注试点地区扩围进展
2. **投向领域创新**：新兴产业基础设施、新质生产力领域
3. **偿债风险管理**：偿债备付金制度建立情况
4. **资金使用效率**：项目开工率和实物工作量形成情况

### 6.3 项目申报建议

| 优先申报领域 | 理由 |
|--------------|------|
| 城市更新 | 政策支持力度大，占比25-30% |
| 产业园区 | 基建主力投向，收益相对稳定 |
| 交通基础设施 | 可作资本金，带动效应强 |
| 土地储备 | 重启发行，窗口期有限 |
